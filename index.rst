FMD documentation
=================

FMD stands for **F**\ ree **M**\ olecular **D**\ ynamics. It is a library written in C for performing classical molecular dynamics simulations. It uses MPI for parallel computation. FMD is free software distributed under the terms of the GNU Public License Version 3 (GPLv3).
